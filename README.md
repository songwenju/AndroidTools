### AndroidTools

---
使用方法：在gradle 文件中添加：

```
compile 'com.songwenju.tools:androidtools:1.0'
```
some android tools for developer

| 类名                       | 作用    |
| --------                  | -----:   |
| DbConstants.java          | 数据库常用的字段常量     | 
| HttpConstants.java        | Http 常用字段常量      | 
| AESUtil                   | AEs 加密工具类      |
